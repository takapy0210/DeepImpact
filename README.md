## Make dataset

学習のためのデータは、csvファイルで用意します

* レース情報：
```
レース名|競馬場|馬場|距離|天気|オッズ|日付
```

* 出走馬情報：
```
着順|馬名|騎手名
```

* csvデータ：
```
レース情報,出走馬情報,出走馬情報,出走馬情報・・・
```

* 履歴データ：

$ cat horse_history.csv
